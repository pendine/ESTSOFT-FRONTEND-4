엄격 모드 : 자바스크립트만 존재한다 봐도 됨 - 최신 변동사항을 적용하는지 마는지 (use strict 가 적용되어있으면 최신 변동사항 적용) - JS의 역사와 관련 - JS는 호환성이슈 0 -> 기존기능 변경이 거의 없었는데 새로운 기능들은 자주 추가됨 - 작성한 코드는 망가지지 않지만 단점이 있었음 - ES5 등장 -> 기존기능 일부 변경 -> 호환성문제 -> 기능이 제대로 동작안함 -> 변경사항 대부분을 기본모드에서는 비활성화되도록설계 -> use strict라는 키워드를 썼을때만 변경사항을 적용할 수 있도록 자바스크립트가 패치. use strict = 엄격모드 활성화 - 어떤 방식으로 구현했는지에 따라 use strict를 적용할지 말지 결정함 -
