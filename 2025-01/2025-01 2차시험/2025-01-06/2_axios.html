axios : 브라우저와 node를 위한 http 비동기 통신 라이브러리 -> 프로미스와 관련된 API(프로미스 API를 활용하는 라이브러리) -> fetch와 유사하지만 프레임워크 사용시 axios 사용 사용하는 이유 보안성 우수 object를 포함할 수 있음 받아오는 데이터를 자동으로 JSON 타입으로 가져옴 => ? 요청 가로채기 가능 다운로드 진행 기본 지원 요청 취소 후 타임아웃 기능 제공 큰 단위의 프로젝트에서 고려해볼만한 라이브러리 -> 간단할때는 fetch 확장성 고려시 axios 사용 방법 자체는 fetch와 유사함 mehtod, url, headers, data(post 방식 http 바디), params(get방식), responseType, withCredentials(쿠키값 전달시 설정)
