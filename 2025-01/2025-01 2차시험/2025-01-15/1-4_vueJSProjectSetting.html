vueJS 프로젝트 세팅 ( CLI ) 1. node JS 설치 필요 - 버전 중요 - 버전 업데이트시 중요사항 ( 더이상 안쓰이는 메서드 존재 할 수 있음) ( node, react, vue는 업데이트가 자주 있음 - 업데이트 후 기존 동작 확인 필수) 2. 터미널 실행 3. 프로젝트를 세팅할 폴더로 이동 4. 커멘드 실행 : npm init vue@latest 5. 세팅 - 여기서는 프로젝트의 이름은 기본값 모든 설정은 no 상위폴더에 vue-project 라고 생성완료 6. vue가 제대로 설치됬는지 확인절차 필요 프로젝트 폴더로 이동 7. 터미널에서 명령어 npm install 사용 <- node js 적용 8. 터미널에서 명령어 npm run dev 사용 <- 실행
